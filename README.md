# RollCallByFace
Đây là đồ án AI nhận diện khuôn mặt áp dụng cho việc điểm danh được thực hiện bởi Lê Dĩ Khang 

Để chạy được dự án vui lòng cày:

** Pyhton 3xxx
** OpenCV
** Pillow
** Và một số thư viện cơ bản khác

-> Để bắt đầu hãy chạy file có tên GUI.py
